## 主な仕様

- **現在準備中**表示：ヒーロー、予約・決済、ご利用の流れ
- **CTA**：Instagram誘導のみ（予約・問い合わせフォームなし）
- **フォント**：Noto Serif JP（見出し）/ Noto Sans JP（本文）/ Cormorant Garamond（英語ラベル）
- **カラー**：白 + 薄緑（#5A7A52）+ 木茶（#6B4A2A）+ グレー系
- レスポンシブ対応（モバイル・タブレット・PC）
- スクロールアニメーション（IntersectionObserver）
- パーラックス（ストーリー背景）
